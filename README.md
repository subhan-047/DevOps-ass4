# Jenkins CI/CD Pipeline using Microsoft Azure

This project demonstrates a simple Jenkins CI/CD pipeline that deploys a static HTML application to a Microsoft Azure Virtual Machine.

## Tools Used
- Git & GitHub
- Jenkins
- Microsoft Azure VM
- Nginx Web Server

## Pipeline Flow
1. Code is pushed to GitHub
2. Jenkins automatically triggers the pipeline
3. Jenkins pulls the latest code
4. Application is deployed to Azure VM
