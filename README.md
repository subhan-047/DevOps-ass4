# Jenkins CI/CD Pipeline using AWS

This project demonstrates a simple Jenkins CI/CD pipeline that deploys a static HTML application to a AWS Virtual Machine.

## Tools Used
- Git & GitHub
- Jenkins
- AWS VM
- Nginx Web Server

## Pipeline Flow
1. Code is pushed to GitHub
2. Jenkins automatically triggers the pipeline
3. Jenkins pulls the latest code
4. Application is deployed to AWS VM
